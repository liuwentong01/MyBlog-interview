学习 Shell 命令笔记
====================

第一天：文件操作
- 学会了 ls、cd、mkdir、rm 等基本命令
- ls -la 可以显示隐藏文件
- mkdir -p 可以递归创建目录

第二天：文本处理
- grep 用于在文件内容中搜索
- find 用于查找文件本身
- awk 适合处理列数据
- sed 适合做文本替换

第三天：管道与重定向
- 管道 | 可以把命令串联起来
- > 覆盖写入，>> 追加写入
- 2>&1 可以合并错误输出

第四天：Git 操作
- git status 查看状态
- git add + git commit 是基本工作流
- git stash 临时保存很方便
