# 猿编程销售话术/客服应对手册 - 总结分析

本文档为猿编程（少儿编程教育机构）销售话术手册的结构化总结，用于辅导老师在面对家长各种异议时的标准化回应。

## 文档结构

共分为 **12大类风险场景**，每个场景下提供"共情 → 反问/了解 → 解决问题 → 迁移编程价值 → 关单"的标准话术流程：

| 序号 | 风险类型 | 核心场景 |
|------|----------|----------|
| 一 | 前期无效用户 | 未添加、未安装、无参课数据 |
| 二 | 学员兴趣/意愿风险 | 孩子不感兴趣、明确抗拒 |
| 三 | 家长认知风险 | 不了解编程价值、无需求 |
| 四 | 时间冲突风险 | 与其他课外班冲突、近期旅游、住校等 |
| 五 | 家长选择偏重风险 | 优先学主科、看重其他兴趣班 |
| 六 | 竞品对比风险 | 对比其他机构（如核桃编程）、倾向线下 |
| 七 | 语言对比风险 | 想学C++而非Python |
| 八 | 家庭经济支持 | 觉得不值/确实没钱 |
| 九 | 决策人差异风险 | 老人带孩子、父母意见不一致 |
| 十 | 决策人主观意愿风险 | 没时间督学、觉得孩子专注力差/坚持不了/学不会/压力大 |
| 十一 | 学业与健康限制 | 视力问题、生病 |
| 十二 | 信任风险 | 怕被骗、退课难 |

## 核心销售策略

1. **共情先行** — 先认同家长顾虑，建立信任
2. **反问定位** — 通过提问精准定位家长真实担忧
3. **价值植入** — 编程=未来必修课、提升学科成绩、逻辑思维培养、考级竞赛升学
4. **降低决策门槛** — 反复强调"29节课试学期"、"全额退款"、"免费赠课"、"免费硬件套装"
5. **制造紧迫感** — 名额有限、优惠活动、先占位再决定

## 关键卖点（反复出现）

- 一周一次课，时间灵活
- 29节课试学期 + 全额退款保障
- 免费智能硬件套装（价值上千）
- 免费考级/竞赛培训
- 双师制（主讲+班主任）
- 招商银行资金监管
- 猿辅导品牌背书、800万学员、清华合作

## 各场景话术要点

### 二、学员兴趣/意愿风险
- 类比英语：编程和30年前的英语一样，是时代必修课
- 强调第一次接触的重要性，避免孩子形成"学不好"的心理定式
- 用硬件套装（乐高同厂教具）和软硬件结合模式吸引孩子兴趣

### 三、家长认知风险
- 引导家长认识编程是学科而非兴趣班
- 政策层面：纳入中小学新课标，浙江高考、湖南中考已考
- 学科融合：一门编程课解决语数外三门补课效果
- 升学路径：科技特长生、强基计划

### 四、时间冲突风险
- 兴趣班冲突：引导家长做"加减法"，用明星案例（安迪）佐证
- 近期没时间：强调正式开课还需一个月，先用赠课过渡

### 五、家长选择偏重风险
- 优先主科：编程与学科相辅相成，"编程是学科知识的试验场"
- 其他兴趣班：强调编程综合性强，4个月可冲刺国家证书

### 六、竞品对比风险
- 线上对比（核桃编程）：从课程体系、直播vs录播、长期规划、隐形消费四个维度对比
- 倾向线下：从互动性、师资、便利性、性价比四个维度说服

### 七、语言对比风险
- 想学C++：建议先学Python打基础（基础相通），高阶再过渡C++

### 八、家庭经济支持
- 有钱不花：详细外化课程价值（75节课含图形化+Python+机器人+考级培训）
- 没钱：提供分期方案（花呗、京东、抖音0元试课通道）

### 九、决策人差异风险
- 老人带孩子：强调编程培养自主学习能力，不需要老人操心
- 父母不一致：让妈妈转发孩子作品给爸爸看，先抢占名额试学

### 十、决策人主观意愿风险
- 三分钟热度/不专注：编程的"输入-思考-调试-成功"闭环培养坚韧品格
- 学不会：课程按年龄分阶设计，服务500万+学员验证
- 学业压力大：编程培养的思维能力反而帮孩子减负

### 十一、学业与健康限制
- 视力问题：影响视力的是用眼习惯而非电子产品本身；30+10+30学习模式含眼保健操

### 十二、信任风险
- 品牌背书：猿辅导旗下、央媒好评、清华合作、百度词条
- 资金安全：招商银行资金监管、支持开发票
- 兜底方案：第三方平台（淘宝）支付
